
# TargetSistemasTest 

Make sure you have [Node.js](http://nodejs.org/) installed.

## Cloning

  

```sh
git clone git@github.com:leohige/TargetSistemasTest.git
```

## Install Dependencies
```
npm install
```

## Run Tests
```
npm test
```

## Building App
```
npm run build
```

## Run App
```
npm start
```
